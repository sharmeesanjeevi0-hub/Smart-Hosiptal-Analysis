🏥 Smart Hospital Analytics Dashboard
📌 Project Description
This project is a Power BI dashboard developed to analyze hospital operations using a synthetic dataset. The dashboard helps monitor patient flow, hospital performance, waiting time, revenue, and patient satisfaction.

🎯 Objectives
Analyze hospital operations

Monitor patient admissions

Track waiting time

Measure patient satisfaction

Improve decision-making using data

📂 Dataset
Synthetic Hospital Dataset

Total Records: 3,000

Format: CSV

💡 Key Insights
Waiting time affects patient satisfaction.

Some departments have more patients than others.

Doctor workload varies by department.

Revenue differs across departments.

📌 Conclusion
This dashboard provides useful insights into hospital operations and helps improve patient care and resource management.
